### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd 8_ChessGame
```

2. Install dependencies:

```bash
npm install
```

### Running the Project

#### Development Mode

```bash
npm run dev
```

The application will be available at: `http://localhost:3000`

The development server will automatically reload the page after each code change (hot reload).

#### Production Build

```bash
npm run build
```

The optimized version will be generated in the `dist/` folder.
