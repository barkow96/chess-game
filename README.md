## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- yarn (recommended package manager)

### Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd 8_ChessGame
```

2. Install dependencies:

```bash
yarn install
# or simply
yarn
```

### Running the Project

#### Development Mode

```bash
yarn dev
```

The application will be available at: `http://localhost:3000`

The development server will automatically reload the page after each code change (hot reload).

#### Production Build

```bash
yarn build
```

The optimized version will be generated in the `dist/` folder.
